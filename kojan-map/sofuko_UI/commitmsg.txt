事業者機能とUI調整を追加修正

- 事業者会員がユーザー名（表示名）を編集できるように `MyPage` に編集UIを追加。保存時に投稿表示にも反映されるよう `MainApp` 内で更新処理を実装。
- 一般会員はユーザー名を持たない方針に変更し、UI上は '匿名' 表示とする（一般会員の名前編集UIは除去）。
- ピンの「通報」フォームを開いている間はリアクション（いいね）を無効化するように変更。
- 事業者会員からのリアクションを禁止するバリデーションを追加（UIで disabled にしています）。
- ピン詳細にブロックボタンを追加。ブロック時はトースト表示を出し、`MyPage` のブロックリストに反映されアンブロック可能。
- 既存 UI コンポーネントの import に含まれていたバージョン接尾辞（例: lucide-react@...）を除去してモジュール解決と型チェックを修正。
- TypeScript の型定義を devDependencies に追加し、`tsconfig.json` を追加して型チェックが通るように修正。
- そのほか軽微な修正（`main.tsx` の拡張子削除など）およびパッケージ再インストール/開発サーバ起動確認。

主な変更ファイル:
- src/components/MyPage.tsx
- src/components/PinDetailModal.tsx
- src/components/MainApp.tsx
- src/App.tsx
- src/main.tsx
- src/components/ui/*（import 修正多数）
- package.json, tsconfig.json

備考:
- 変更は現時点ではフロントエンドのローカル状態で動作します（バックエンド永続化は未実装）。
- ブロック済みユーザーの投稿を一覧から自動的に除外する処理はフォローアップで対応予定です。
